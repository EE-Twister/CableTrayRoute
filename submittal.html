Submittal Package Generator

About Submittals

A submittal package is a formal project deliverable submitted to engineers of record, owners, or authorities having jurisdiction (AHJ) for review and approval. It typically includes project information, equipment and cable schedules, raceway data, tray fill calculations, and code compliance declarations.

This tool compiles your project data into a formatted, print-ready document. Use Print / Save as PDF to generate the final submittal file.

Code references: NEC (NFPA 70), NEMA VE 1 (cable tray), IEEE Std 835 (ampacity).
