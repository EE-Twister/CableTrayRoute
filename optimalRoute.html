Pull strategy

A hand-pull recommendation is a planning instruction, not a crew-capacity certification. Confirm staffing, access, grip method, cable control, and site safety.

Field equipment working limits

Zero reel payoff tension uses a 25 × cable-weight estimate. The cable, puller, rope, grip, and anchorage are checked as a weakest-link system.

Method & Assumptions

Optimal paths are computed using Dijkstra where C = Σ w_i.

Variables:

  • w_i - weight of segment i
  • C - total path cost

Citations: Shortest path algorithms

Applicability: non-negative weights and connected graphs.

See also: Load Calculation Formulas, Cable Fill Rules, Routing Assumptions.
