Auto-Size Equipment

Automatically select conductor sizes, overcurrent device ratings, and transformer kVA per NEC 2023. Covers feeders, branch circuits, motor circuits, and transformer protection.

Method — NEC 210.20 / 215.3 / 240.4 / 310.15 / 392.80

For continuous loads (≥3 hours): conductor ampacity and OCPD rating must be at least 125% of the load current (NEC 210.20(A), 215.3).

Conductor size is selected from NEC Table 310.15(B)(16). Ambient temperature, bundling, and cable tray correction factors per NEC 310.15(B)(1)(a), 310.15(C)(1), and 392.80(A) are applied based on the Installation Conditions you enter below.

OCPD rating is the next standard rating from NEC 240.6(A) at or above the conductor's ampacity (NEC 240.4(B)).

Method — NEC 430 / 310.15 / 392.80 (Motor Branch Circuits)

Full load current (FLC): from NEC Table 430.250 (3-phase) or Table 430.248 (single-phase). Note: always use the NEC table values, not the motor nameplate current, for sizing branch circuit components.

Branch circuit conductor: ≥ 125% of motor FLC (NEC 430.22(A)). Ambient temperature, bundling, and cable tray correction factors per NEC 310.15(B)(1)(a), 310.15(C)(1), and 392.80(A) are applied based on the Installation Conditions you enter below.

Branch circuit OCPD (inverse time breaker): ≤ 250% of FLC per NEC Table 430.52. Next standard OCPD size is selected.

Overload relay: set at 115% of FLC for motors with service factor ≥ 1.15 or temperature rise ≤ 40°C, otherwise 125% (NEC 430.32(A)(1)).

Method — NEC 450.3(B) (Transformers ≤ 600V)

Transformer kVA: next standard kVA rating at or above the required load.

Primary OCPD: ≤ 125% of primary rated current (or 167% if primary current ≤ 9A) per NEC Table 450.3(B). Next standard OCPD size is selected.

Secondary OCPD: ≤ 125% of secondary rated current per NEC 450.3(B).

Secondary conductors: sized at 125% of secondary rated current (continuous load per NEC 310.15(B)(16)).

Auto-Size Equipment Help

This tool applies NEC 2023 rules to automatically size conductors, overcurrent protection devices (OCPD), and transformers.

Feeder / Branch Circuit

Enter the load current (or kW/PF), check if the load is continuous, and select conductor material and temperature rating. The tool selects the smallest NEC-compliant conductor from Table 310.15(B)(16) and the next standard OCPD from NEC 240.6(A).

Motor Branch Circuit

Enter motor HP, voltage, and phase. The tool looks up NEC table FLC values and applies NEC 430.22 (conductor) and NEC 430.52 (breaker) rules automatically.

Transformer

Enter required kVA, primary voltage, and secondary voltage. The tool selects the next standard transformer kVA and sizes primary and secondary overcurrent protection per NEC 450.3(B), Table 450.3(B).
