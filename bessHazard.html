BESS Hazard / Thermal Runaway Screening

Screen layout distances, generic thermal-runaway sensitivity, and preliminary deflagration-vent inputs before completing the project-specific code, test-report, and engineering review.

Method & Standards

Advisory separation screening: The following built-in values are conservative review triggers only. They are not represented as NFPA 855 minimum clearances; the actual requirement depends on installation type, adopted code edition, listing, UL 9540A results, protection features, and AHJ decisions.

Exposure type≤ 50 kWh system> 50 kWh system

Ignition / electrical source0.9 m (3 ft)0.9 m (3 ft)

Generic propagation estimate: A lumped thermal-mass sensitivity model estimates cell-to-cell, module, and rack timing. UL 9540A is a test method; this calculation is not a UL 9540A result and cannot predict a listed system's performance.

Preliminary deflagration-vent screening: A Bartknecht-form equation estimates an area from enclosure volume, Pstat, and generic chemistry-wide KG and Pmax assumptions:

Av = (Pstat−0.5682 × KG0.5922 × V0.6672 × Pmax0.1723) / 1640

Important: This screen cannot determine NFPA 855 or NFPA 68 compliance, approve an HMA, or replace UL 9540A and project-specific gas data. Final design requires the adopted code, listed-system reports, a qualified engineer, and AHJ review.

BESS Hazard / Thermal Runaway Help

This page performs preliminary BESS hazard screening. It does not complete or approve a Hazard Mitigation Analysis and does not determine code compliance.

Inputs

  • Rated capacity (kWh): Total rated energy of the ESS installation (sum of all BESS units).
  • Chemistry: Selects generic KG, Pmax, and propagation assumptions for sensitivity screening only.
  • Cells/modules per rack: Used by the generic propagation sensitivity model; enter the actual listed-system arrangement.
  • Room volume: Net internal volume of the ESS room used for deflagration vent sizing.
  • Pstat: Vent panel opening (activation) pressure, typically 3–10 kPa for lightweight panels.
  • Installed vent area: Total area of explosion-relief panels. Set to 0 if none are provided.
  • Exposures: Add property lines, occupied buildings, or ignition sources for comparison with advisory review distances.

Results

  • Separation screening: Flags distances below a generic advisory reference. It is not a code-compliance check.
  • Propagation estimate: Shows generic sensitivity timing, not UL 9540A test performance.
  • Deflagration-vent screening: Compares installed area with an equation result based on assumed gas properties; final NFPA 68 design needs project data.
  • Status: Always requires engineering review. Screening alerts identify inputs that need prompt investigation, not code pass/fail decisions.
