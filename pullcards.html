Cable Pull Cards & Pull Table

Method & Assumptions

Cables are grouped into pulls — sets of cables that share the same cable type (Power, Control, Signal) and traverse the same ordered sequence of raceway segments. Cables in the same pull can be pulled simultaneously by field crews.

Pull tension is estimated using the capstan equation for bends and a constant friction coefficient (μ = 0.35) for straight runs. Combined cable weight per foot is used for multi-cable pulls.

References: NEC Article 300.31, IEEE Std 1185, AEIC CG5.

See also: Optimal Route, Cable Tray Fill.

Pull Cards Help

This tool generates cable pull cards and a pull table from routing results.

Pull table: Groups cables by type and shared route path. Cables of the same type traversing the same raceways in the same order are combined into a single pull for simultaneous installation.

Pull cards: Click any row to view a detailed pull card showing the step-by-step route, cable list, tension estimate, and physical properties.

Import: Use the route_data.xlsx file from the Optimal Route page, or load directly from the current project if routes have already been computed.
