# CableTrayRoute Revit Connector Starter

This is a source template for an Autodesk Revit add-in. It is not compiled by CableTrayRoute CI.

## Prerequisites

- Autodesk Revit 2026 or project-selected target version.
- Revit API assemblies available from the local Revit installation.
- .NET Framework target compatible with the selected Revit version.

## Install

Copy `CableTrayRoute.RevitConnector.addin` to one of:

- `%APPDATA%/Autodesk/Revit/Addins/2026/`
- `%PROGRAMDATA%/Autodesk/Revit/Addins/2026/`

Update the `Assembly` path in the `.addin` file after building the DLL.

## Commands

- `ExportCableTrayRouteJson`
- `ImportCableTrayRoutePreview`
- `ValidateCableTrayRoutePackage`
- `OpenCableTrayRouteBridge`

All commands are review-only placeholders. Add project-specific authoring updates only after validating the connector package in CableTrayRoute.

## Bridge Contract

The starter targets `bim-connector-contract-v1` and the Revit bridge readiness package generated by `analysis/revitConnectorBridge.mjs`.

- File exchange is the default supported workflow.
- The optional local HTTP bridge endpoint is documented as `http://localhost:41731/cabletrayroute/revit-bridge`.
- Import preview must validate the connector JSON before any Revit element creation or update code is enabled.
- CI for CableTrayRoute validates this source text and JSON contract compatibility, but does not compile against Autodesk DLLs.
