Voltage Stability Analysis

Method & Standards

This study sweeps the total system load from the operating point up to a user-specified maximum (the load scaling factor λ) and solves a Newton-Raphson power flow at each step. The resulting plot of bus voltage vs. system load is the P-V nose curve. The last converged load point is the maximum loadability; the margin from the operating point to that limit is the loadability margin (MW).

The companion Q-V curve sweeps reactive power injection on a selected bus from negative (absorbing) to positive (injecting) and records the resulting voltage. The minimum Q injection at which the system still converges defines the reactive power margin (MVAR).

Standards: NERC TPL-001-5 (Transmission Planning Performance Requirements), IEEE Std 2800-2022 (voltage stability screening), WECC Voltage Stability Methodology.

Algorithm: Sequential Newton-Raphson power flows with load parameterization. The step-size is fixed; the collapse point is identified as the first divergent step. For production-grade continuation power flow (CPF) through the nose, the arc-length parameterization in the full CPF module should be used.

Bus model: Slack (swing) bus holds voltage magnitude and angle; PV buses hold voltage magnitude and active injection; PQ buses hold active and reactive load.

System Base

Common values: 100 MVA (transmission), 10 MVA (industrial). All per-unit calculations use this base.

Buses

Define at least one slack bus and one PQ load bus. Base kV applies to that bus's per-unit conversion.

Branches

Enter series impedance in ohms. Each branch is added once; the admittance is reflected at both endpoints automatically.

P-V Curve Parameters

The sweep continues until the power flow diverges or λ reaches this value. A value of 3.0 means 300% of base load.

Smaller steps give a smoother curve and more accurate nose-point detection but increase compute time.

Q-V Curve Parameters

The bus at which reactive injection is swept. Leave blank to use the first PQ bus automatically.

Voltage Stability Help

P-V Curve (Nose Curve)

The P-V curve plots the voltage at each bus as total system load is scaled from the operating point. As load increases, voltage droops. Beyond the nose point, the Newton-Raphson power flow diverges, indicating voltage collapse. The loadability margin is the MW distance from the operating point to the nose.

Q-V Curve

The Q-V curve shows how much reactive power must be injected at a bus to maintain a given voltage. The minimum of the Q-V curve (most negative injection still achieving convergence) defines the reactive margin. A positive reactive margin means the system can absorb more reactive load before collapse.

NERC TPL-001 Screening

NERC TPL-001-5 requires planning-event voltage stability analysis for Category P5–P7 contingencies. A commonly used screening threshold is a loadability margin ≥ 5% of operating load.

Bus Types

  • Slack: Reference bus. Voltage magnitude and angle are fixed (typically 1.0 pu, 0°). Supplies whatever power is needed to balance the system.
  • PQ: Load bus. Active (P) and reactive (Q) load are specified; voltage floats.
  • PV: Generator bus. Active injection (P) and voltage magnitude are specified; reactive output floats within limits.
