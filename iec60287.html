IEC 60287 Cable Ampacity — Thermal Circuit Model

Method & Assumptions

Cable current rating follows the steady-state thermal circuit method of IEC 60287-1-1:2023 (Calculation of the current rating — current rating equations for 100% load factor and calculation of losses). Applicable to three-phase AC power cables in any installation condition.

Current rating formula (IEC 60287-1-1 §3.1.1):

I = √{ [Δθ − W_d·(0.5·T1 + n·(T2+T3+T4))] / [R_ac·(T1 + n·(1+λ1)·T2 + n·(1+λ1+λ2)·(T3+T4))] }

where Δθ = θ_max − θ_ambient (K), T1–T4 are thermal resistances (K·m/W), R_ac is the AC conductor resistance at operating temperature (Ω/m), n is the number of load-carrying conductors, and λ1, λ2 are sheath/armour loss factors.

AC conductor resistance (IEC §2.1):

R_ac = R_dc_θ × (1 + y_s + y_p)
R_dc_θ = R_20 × [1 + α·(θ − 20)]

where y_s and y_p are skin and proximity effect coefficients per IEC §2.1.2–2.1.3.

Soil external thermal resistance T4 — Kennelly formula (IEC 60287-2-1 §2.2.1):

T4 = (ρ_s / 2π) × ln[u + √(u² − 1)]   where u = 2L / D_e

L = burial depth to cable centre (m), D_e = cable outer diameter (m), ρ_s = soil thermal resistivity (K·m/W).

Grouping derating (IEC 60287-2-1 Table 1): applied when multiple cables share the same installation route. Mutual heating reduces individual cable ratings.

Conductor

IEC 60228 Class 2 stranded conductor. R₂₀ values per IEC 60228 Table 1.

Copper: higher conductivity, smaller cross-section for same current. Aluminium: lower cost and weight for distribution cables.

Number of current-carrying conductors in the cable. For three-phase systems use 3. The neutral conductor is not counted unless it carries significant current.

Insulation

The maximum conductor operating temperature determines the thermal budget Δθ = θ_max − θ_ambient. XLPE and EPR are standard for modern distribution and transmission cables.

Used to look up the default IEC 60502-1 nominal insulation thickness. Override the thickness below if you have data-sheet values.

Nominal insulation wall thickness per IEC 60502-1 or cable data sheet. Click Use Default to look up the IEC 60502-1 value for the selected conductor size and voltage class.

Outer jacket/sheath thickness. Use 0 if no outer sheath. Typical values: 2–4 mm for distribution cables.

Check if the cable has steel-wire armour. Armour contributes additional losses (λ2 ≈ 0.06) and adds approximately 6 mm to the cable outer diameter.

Installation Method

Determines the external thermal resistance T4. Direct burial in good soil (ρ ≤ 1.0 K·m/W) often gives higher ratings than free air due to the large thermal mass of soil.

Depth from ground surface to the cable centre-line (not top of cable). IEC reference depth is 0.8 m (800 mm). Greater depth increases T4 and reduces rating.

Thermal resistivity of the surrounding soil. The IEC reference value is 1.0 K·m/W (moist loam). Typical ranges: well-irrigated/clay 0.5–0.8; average loam 1.0–1.5; dry sand/gravel 2.0–3.0; rock 1.5–2.5. Measure per IEEE 442 / IEC 60287-3-2 for critical projects. Consider thermal backfill (ρ ≤ 0.8 K·m/W) for high-load cables.

Outside diameter of the buried conduit or duct. Used to calculate the external soil thermal resistance using Kennelly's formula.

Ambient & System Conditions

Ground (soil) or air temperature at the installation site. IEC reference: 20 °C for direct burial, 30 °C for free air. Use the maximum expected seasonal temperature for conservative sizing. Must be less than the insulation maximum temperature.

Affects skin and proximity effect coefficients. Higher frequency increases AC resistance slightly.

Phase-to-earth operating voltage. Used to calculate dielectric losses W_d. For cables rated ≤ 36 kV (U₀ ≤ 18 kV), dielectric losses are negligible and this value has no effect. Enter 0 for LV/MV distribution cables.

Cable Grouping (Optional)

Multiple cables in close proximity run hotter due to mutual heating. Leave at 1 cable for a single cable or if cables are widely spaced.

Total number of cables sharing the same installation route (e.g. same trench or tray). Grouping factors per IEC 60287-2-1 Table 1 are applied automatically.

Flat spaced gives the best heat dissipation. Touching cables run hotter. Trefoil touching is common for three-phase single-core cable sets.

IEC 60287 Cable Ampacity Help

This tool calculates the steady-state current rating of a power cable using the IEC 60287-1-1:2023 thermal circuit model.

Workflow

  1. Select the conductor cross-section and material. The tool looks up the IEC 60228 DC resistance at 20 °C automatically.
  2. Choose the insulation type and voltage class, then click Use Default Thickness to populate the IEC 60502-1 insulation wall thickness, or enter your data-sheet value manually.
  3. Select the installation method and enter burial depth and soil thermal resistivity for direct-burial or conduit installations.
  4. Set the ambient temperature and frequency.
  5. For cable grouping, enter the number of cables and arrangement.
  6. Click Calculate Ampacity to see the rated current and thermal breakdown.
