Cable Tray Support Span Calculator

Method & Assumptions

Maximum support span is governed by the L/100 deflection limit defined in NEMA VE 1-2017 §4. For a uniformly distributed cable load on a simple span, midpoint deflection is:

δ = 5wL⁴ / (384EI)  ≤  L/100

Normalising against the tray's rated conditions at the class test span gives:

max_span = L_rated × (w_rated / w_actual)^(1/3)

Variables:

  • w_actual — total cable weight per linear foot of tray (lbs/ft)
  • w_rated — NEMA load-class working load (lbs/ft)
  • L_rated — class test span (12 ft for all standard classes)

Citations: NEMA VE 1-2017

Applicability: aluminum and steel ladder trays, solid-bottom trays, and wire-mesh trays with standard NEMA load classes. Fiberglass trays (NEMA FG 1) use identical load-class designations and the same formula.

See also: Cable Tray Fill, Raceway Schedule.

Tray Load Class

Select the load class stamped on the tray or specified by the manufacturer. Class 16A (16 lbs/ft, 12 ft span) is typical for industrial installations.

Support Span Help

This tool calculates the maximum cable tray support span per NEMA VE 1 using the L/100 deflection criterion.

Manual mode: build a cable list or enter a total weight directly, then click Calculate Span.

Schedule mode: reads trays and cables from the current project and evaluates every tray at once.
