Method & Standards

Bend geometry is calculated using standard multiplier and shrink tables (Tom Henry / Mike Holt). NEC 358.24 limits total bends to ≤ 360° between pull points for EMT, RMC, IMC, and LFMC.

Conduit Bend Schedule Help

Click + Add Conduit Run to enter a conduit run. For each run specify the conduit trade size and add bends (90°, offset, kick, or saddle) with their dimensions.

NEC 358.24: The total degrees of all bends between pull points must not exceed 360°. A red badge appears if a run violates this limit.

Bend geometry: 90° — enter stub-up height. Offset — enter the height to move and choose the bend angle. Kick — enter the kick height. Saddle — enter the obstacle height.

Pull-Box Sizing (NEC 314.28): Click + Add Pull Point to size a pull box. Choose straight or angle/U pull and enter the conduit trade sizes on each wall.

Click Calculate to run geometry and NEC checks. Use Export CSV to download the full bend schedule.
