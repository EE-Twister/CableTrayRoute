Component Library

Define components for the one-line palette.

The one-line editor loads component definitions from componentLibrary.json. Each object in this array describes a subtype available in the palette.

[{
  "subtype": "MLO",
  "label": "MLO",
  "icon": "icons/MLO.svg",
  "category": "panel",
  "ports": [{ "x": 0, "y": 20 }, { "x": 80, "y": 20 }],
  "schema": [
    { "name": "voltage", "label": "Voltage", "type": "number" }
  ]
}]

Fields:

  • subtype – unique identifier for the component.
  • label – text shown in the palette.
  • icon – path to an SVG icon.
  • category – group for palette organization (e.g. panel, equipment, load).
  • ports – connection points relative to the component width and height.
  • schema – array of property descriptors with name, label, and type.

Add new components by appending objects to the JSON array and providing corresponding icons. The application reloads the file at startup, so no JavaScript changes are required.
