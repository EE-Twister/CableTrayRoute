Quick Start

Follow the six-step workflow to plan and route cables efficiently. Each tool can operate on its own, but completing the steps in sequence yields the best results.

  1. Cable Schedule – define the cables you need to route.
  2. Raceway Schedule – catalog trays, conduits, and ductbanks.
  3. Ductbank – analyze underground ductbanks and thermal limits.
  4. Tray Fill – visualize how cables occupy trays.
  5. Conduit Fill – evaluate stand‑alone conduit loading.
  6. Optimal Cable Route – generate the best path for each cable.

    The Route Breakdown lists any conduits skipped during routing along with the reason.

You can open any tool directly for ad‑hoc checks, but stepping through them in order ensures data flows smoothly from one phase to the next.
