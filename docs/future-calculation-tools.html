Future Calculation Tools

Scoping guidance for potential engineering calculators requested by users.

What it takes to ship reliable calculators

Each topic can be delivered as a practical engineering tool if the workflow is split into: (1) required inputs, (2) transparent calculation method, (3) code/standard references, and (4) a report-ready output page with assumptions clearly shown.

This document provides a scoped MVP approach for Heat Trace Sizing, Cathodic Protection, and Dissimilar Metal Corrosion checks.

1) Heat Trace Sizing

Minimum input data model

  • Pipe/tray geometry (diameter, length, fittings, supports, elevation profile).
  • Insulation system (material, thickness, thermal conductivity, weather barrier).
  • Process constraints (minimum maintain temperature, startup temperature, ambient design minimum).
  • Electrical limits (available voltage, circuit length limits, breaker type, hazardous area class/division zone).
  • Operational profile (continuous maintain, intermittent freeze protection, startup duty cycle).

Core calculations to implement

  • Steady-state heat loss by segment (straight run + fitting correction factors).
  • Startup heat-up load check (mass of process + pipe + insulation lag effects).
  • Circuit selection logic for constant watt vs. self-regulating cable families.
  • Electrical validation (voltage drop, max circuit length, current, breaker utilization).

MVP output

  • Recommended cable type/watt density by segment.
  • Segmented bill of material (cable length, terminations, power connection kits).
  • One-page design basis with assumptions and pass/fail checks.

2) Cathodic Protection

Minimum input data model

  • Asset geometry and coating condition (surface area, coating efficiency, defect factors).
  • Electrolyte/soil data (resistivity by layer, moisture, pH, chloride/sulfate where available).
  • Design life, protection criteria, and operating environment.
  • Current drainage/interference sources (nearby DC systems, rail, rectifiers, foreign structures).

Core calculations to implement

  • Design current density and total protection current demand.
  • Anode bed sizing (impressed current and sacrificial options).
  • Anode consumption/life estimation and rectifier range checks.
  • Voltage drop and attenuation checks for long structures.

MVP output

  • Recommended CP method (sacrificial vs. impressed current) with rationale.
  • Anode quantity, spacing, and expected life range.
  • Monitoring plan fields (test station interval, baseline potentials, acceptance limits).

3) Dissimilar Metal Corrosion Risk

Minimum input data model

  • Base and fastener alloys (e.g., aluminum tray with stainless hardware grade).
  • Area ratio and contact geometry (cathode-to-anode ratio has major impact).
  • Environment class (indoor dry, coastal, chemical washdown, buried, splash-zone).
  • Barrier strategy (isolation washers, coatings, sealants, anti-oxidant compounds).

Core calculations to implement

  • Galvanic potential difference scoring by alloy pair.
  • Electrolyte severity multiplier based on environment exposure.
  • Area-ratio risk amplification and coating effectiveness derating.
  • Estimated corrosion-rate banding (low/medium/high risk, not false precision).

MVP output

  • Risk rating with confidence band and key drivers.
  • Mitigation recommendations prioritized by impact/cost.
  • Inspection interval guidance tied to environment severity.

Implementation roadmap

  1. Define calculation assumptions and reference equations in a standards-backed spec.
  2. Build a shared material/environment library so all three tools reuse one source of truth.
  3. Add transparent report export that lists equations, inputs, and safety margins.
  4. Pilot with sample projects from users to calibrate defaults and reduce bad assumptions.

A realistic first release is a decision-support calculator with clear uncertainty flags, followed by deeper modeling once field feedback is collected.
