Required Geometry Fields

Structure geometry data correctly to avoid skipped entries.

Ductbank records must provide either an outline array of 3D points or both start and end coordinates using the fields start_x, start_y, start_z, end_x, end_y, and end_z.

Conduit records must include a path array with at least two [x, y, z] points describing the run of the conduit.

Entries missing these fields cannot be placed in the model and will be skipped.
