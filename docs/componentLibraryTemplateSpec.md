# Component Library Workbook Template Spec

This document defines the spreadsheet format accepted by Library Manager import (`.xlsx`/`.xls`) and generated by **Download Template**.

## Workbook Sheets

| Sheet | Required | Purpose |
|---|---|---|
| `Components` | Yes | Primary component records. |
| `Categories` | No | Allowed category values. |
| `Icons` | No | Icon key/path mapping. |
| `Ports` | No | Flattened port counts by subtype. |
| `Schema` | No | Flattened JSON schema overrides by subtype. |

## `Components` Sheet

Minimum useful columns:

- `subtype` (required)
- `label` (required)
- `category` (recommended; required by validator if saving)
- `icon` (recommended; required by validator if saving)
- `ports` (optional number)
- `schema` (optional JSON object string)

Example rows:

| subtype | label | category | icon | ports | schema |
|---|---|---|---|---:|---|
| `MCC` | Motor Control Center | equipment | icons/components/MCC.svg | 2 | `{"voltage":{"label":"Voltage","type":"number"}}` |
| `UPS` | UPS | equipment | icons/components/UPS.svg | 2 | `{"kva":{"label":"kVA","type":"number"}}` |

## `Categories` Sheet

Accepted columns (first match used):

- `category`
- `name`
- `value`

## `Icons` Sheet

Accepted columns:

- `icon` or `key`
- `path` or `iconPath`

## `Ports` Sheet (Optional)

Accepted columns:

- `subtype` or `component` or `componentSubtype`
- `ports` or `portCount` or `count` or `value`

## `Schema` Sheet (Optional)

Two supported shapes:

1. Explicit JSON object per subtype (`schema` column).
2. Flattened field rows (`field/name`, `label`, `type`, `required`).

Accepted subtype columns:

- `subtype` or `component` or `componentSubtype`

## Validation Notes

- Import builds a normalized `{ categories, components, icons }` payload and runs library validation before save.
- Duplicate subtype values are rejected at validation time.
- Category values referenced by components but missing from `Categories` generate warnings.
- Invalid JSON in `schema` cells is ignored for that row unless valid fallback fields are present.
