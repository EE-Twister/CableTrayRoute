Troubleshooting

Solutions to common issues.

  • Import errors: Ensure CSV headers match the templates exactly.
  • No routes found: Check that raceways connect between cable endpoints.
  • Incorrect ampacity: Verify soil resistivity and conductor properties.
  • Browser performance: Large projects run faster in a modern desktop browser.
  • Load flow will not converge: Ensure transformer windings are mapped to buses with matching base kV. If the high-side bus is accidentally wired to the low-side winding (or vice versa), the solver drives the secondary voltage toward zero and stalls near the iteration limit.
  • Can't locate a One-Line node from Fix-it Hints: Run Validate so the Fix-it panel stays open, then click the hint entry to jump the canvas to the flagged device. Use the Find search box to select devices by tag when you only have an identifier, and open the properties drawer. The Nodes category lists orphaned nodes so you can inspect their connections or remove the node directly from the menu.
