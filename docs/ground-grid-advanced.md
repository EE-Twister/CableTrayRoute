# Advanced Ground Grid Analysis — Methodology and Limitations

## Overview

This document describes the advanced ground grid features added in Gap #74: two-layer
soil model fitting from Wenner measurements, irregular polygon grid geometry, a
screening-level hazard map, and an inspection risk-point table.

The base IEEE 80 rectangular screening workflow is unchanged. The new tabs extend it
with higher-fidelity inputs and a richer result set, while clearly marking results as
"screening-level" so users know when to escalate to a specialist tool.

## Soil Model Tab

### Wenner Four-Electrode Method

The Wenner array uses four equally spaced surface electrodes (spacing `a`). The two
outer electrodes inject current; the two inner electrodes measure voltage. The apparent
resistivity is:

```
ρa = 2π a (V/I)
```

By repeating the measurement at increasing spacing `a`, deeper soil layers contribute
more to the reading. A typical field survey covers spacings from 0.5 m to 30 m.

### Two-Layer Soil Model (Sunde Image Method)

CableTrayRoute fits a two-layer model: a top layer of resistivity ρ₁ and thickness `h`,
overlying a half-space of resistivity ρ₂.

The forward model uses Sunde's image method (N = 8 terms):

```
ρa(a) = ρ₁ × [1 + 4 × Σ_{n=1}^{N} Kⁿ × ( 1/√(1+(2nh/a)²) − 1/√(4+(2nh/a)²) )]
```

where the reflection coefficient `K = (ρ₂ − ρ₁) / (ρ₂ + ρ₁)`.

**Limiting behaviour:**
- `a → 0`: ρa → ρ₁ (only top layer seen)
- `a → ∞`: ρa → ρ₂ (bottom half-space dominates)
- `ρ₁ = ρ₂`: ρa = ρ₁ for all `a` (homogeneous soil)

### Fitting Algorithm

The fit minimises the RMS relative error:

```
RMSE = sqrt( mean( ((ρa_fitted − ρa_measured) / ρa_measured)² ) ) × 100 %
```

**Step 1 — Coarse grid search**: 7 × 7 × 7 logarithmically spaced grid over:
- ρ₁ ∈ [10, 10 000] Ω·m (7 steps)
- ρ₂ ∈ [10, 10 000] Ω·m (7 steps)
- h  ∈ [0.5, 25] m (7 steps)

**Step 2 — Nelder-Mead simplex refinement** starting from the best grid-search point,
operating in log-space (log ρ₁, log ρ₂, log h) to enforce positivity and improve
scaling.

At least 3 measurements are required. 6–10 well-distributed spacings are recommended.

### Interpreting Results

| Fit Error | Interpretation |
|-----------|---------------|
| < 5 %     | Excellent — two-layer model is a good representation |
| 5–15 %    | Acceptable — minor lateral variation or measurement noise |
| 15–30 %   | Fair — consider three-layer or anisotropic model |
| > 30 %    | Poor — data may have lateral heterogeneity or outliers |

The **"Use in Analysis" button** populates the IEEE 80 tab with ρ₁ (the top-layer
resistivity that most governs surface potential gradients for shallow grids). For
very deep grids (burial depth > h), consider using a weighted effective ρ.

### Standards Reference

- ASTM G57 — Field Measurement of Soil Resistivity Using the Wenner Four-Electrode Method
- IEEE 81-2012 — Guide for Measuring Earth Resistivity, Ground Impedance, and Earth Surface Potentials
- Sunde, E.D. (1968) *Earth Conduction Effects in Transmission Systems*

## Irregular Grid Tab

### Polygon Geometry Editor

The polygon editor allows the user to define a non-rectangular grounding grid by
clicking vertices on the SVG canvas. Vertices can be dragged after placement. The
"Delete" button removes the last vertex.

**Polygon requirements:**
- At least 3 vertices
- Vertices should be specified in order (clockwise or counter-clockwise)
- Self-intersecting polygons are not supported

### Interior Conductor Mesh

Interior conductors are generated by scan-line fill at the specified mesh spacing.
Horizontal runs are clipped to the polygon boundary using ray-casting. Vertical runs
are similarly clipped.

The total conductor length reported in the KPI cards is the sum of all segment lengths
(perimeter + interior mesh).

### Running the Screening Analysis

The **"Run Screening"** button calls `analyzeIrregularGrid()`, which:

1. Computes the polygon area via the shoelace formula
2. Approximates the effective grid side length as `√area`
3. Applies the IEEE 80 mesh voltage, step voltage, and grid resistance formulas using
   the actual conductor length from the polygon geometry

**Limitation:** The irregularity correction factor (`Ki = 0.644 + 0.148 × n`) in
IEEE 80 §B.4.2 was derived for rectangular grids. For markedly non-rectangular shapes
the mesh voltage estimate carries additional uncertainty (±20–30%).

### Standards Reference

- IEEE 80-2013 §B.4 — Grid resistance and mesh/step voltage formulas
- IEEE 80-2013 Annex B — Sverak simplified formula for grid resistance

## Hazard Map Tab

### What the Hazard Map Shows

The hazard map divides the bounding box of the ground grid into a uniform grid of cells
at the chosen resolution (coarse 5 m / standard 2 m / fine 1 m). Each cell is
colour-coded by risk class:

| Colour | Risk Class |
|--------|-----------|
| Green  | Safe — both touch and step within tolerable limits |
| Yellow | Touch risk — touch voltage exceeds tolerable limit |
| Orange | Step risk — step voltage exceeds tolerable limit |
| Red    | Both risks — touch and step both exceed limits |

### Surface Potential Estimation

**This is a screening-level estimate, not a full finite-element or boundary-element
solution.**

Each cell's surface potential is estimated as:

```
V(x,y) = Σ_conductors [ (ρ × Ig) / (2π × L_total) × 1/d_i ]  (normalised)
```

where `d_i` is the distance from (x,y) to the midpoint of conductor segment *i* and
`L_total` is the total conductor length. The result is scaled to span from 0 to GPR.

Touch voltage at a point is approximated as `GPR − V(x,y)`. Step voltage is the
potential difference between (x,y) and (x + 1 m, y).

This simplified superposition:
- Treats the conductor system as an array of point sources
- Does not account for mutual inductance or ground surface reflections
- Does not solve the boundary-value problem on the conductor surface
- Overestimates the hazard zone width in some configurations

**Use SES CDEGS or XGSLab for final design verification.**

### Resolution Trade-offs

| Setting | Cell size | Grid cells (30×30 m) | Typical compute |
|---------|-----------|----------------------|-----------------|
| Coarse  | 5 m       | 36                   | < 0.1 s         |
| Standard| 2 m       | 225                  | < 0.5 s         |
| Fine    | 1 m       | 900                  | 1–3 s           |

### Exporting the Map

- **Export SVG**: Saves a vector SVG file with embedded colour cells and legend
- **Export PNG**: Renders to a 2× resolution canvas for high-DPI printing

### Disclaimer

> **Screening-level estimate (simplified superposition). Grid conductor influence is
> modelled as superimposed point sources normalised to GPR. This is NOT a finite-element
> or boundary-element solution. Hazard zone boundaries may shift by 1–3 m versus a full
> numerical solution. Use SES CDEGS, XGSLab, or ETAP Ground Grid Systems for final
> design verification and permitting.**

## Risk Points Tab

### Adding Inspection Points

Click **"Add Inspection Point"** and enter X/Y coordinates (in metres, same reference
frame as the polygon geometry). A label (e.g. "Fence gate" or "Transformer pad") helps
identify the point in the exported report.

### How Risk Metrics Are Computed

Each inspection point is matched to the nearest cell in the most recent hazard map
calculation. The displayed values are:

- **Touch V**: Touch voltage at that cell (V)
- **Step V**: Step voltage at that cell (V)
- **Risk Class**: `safe` / `touch-risk` / `step-risk` / `both-risk`
- **Touch Margin %**: `(tolerable − actual) / tolerable × 100` (negative = overstress)
- **Step Margin %**: Same formula for step voltage

### CSV Export

The **"Export CSV"** button downloads a comma-separated file with all inspection points,
suitable for pasting into project risk registers or study reports.

## Known Limitations

| Limitation | Impact | When to escalate |
|------------|--------|-----------------|
| Simplified superposition hazard map | ±20 % on hazard zone extent | Always confirm with CDEGS/XGSLab before construction |
| Two-layer soil only | Multi-layer / anisotropic soils not fitted | Sites with deep clay or weathered rock profiles |
| Polygon irregularity factor from IEEE 80 rectangular basis | ±25 % on Em for L-shaped or T-shaped grids | Grids with aspect ratio > 4:1 or concave shapes |
| No transferred voltage check | Risk from remote earthing not assessed | Sites near pipelines, fences, or remote neutrals |
| No fault-current distribution modelling | Ig is user-supplied | Must ensure Ig includes all parallel earth-return paths |

## When to Use a Specialist Tool

CableTrayRoute's ground grid module is a **screening tool**. Escalate to SES CDEGS,
XGSLab, or SKM GroundMat when any of the following apply:

- Fault current > 10 kA or grid area > 5 000 m²
- Non-rectangular / irregular site with significant clearance constraints
- Hazard map shows any red (both-risk) cells in occupied areas
- Soil has more than two distinct layers or strong lateral variation (fit error > 20 %)
- Permitting or utility interconnection requires a sealed grounding study
- Transferred voltage from adjacent grounded structures must be assessed
- Step/touch exceedance found in screening — escalate before adding mitigation
