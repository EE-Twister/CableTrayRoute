LRFD Load Combinations — Cable Tray Support Design

Method & Assumptions

Factored load combinations per ASCE 7-22 §2.3.2 (LRFD). For cable tray support structures, live load (L), roof live load (Lr), and rain load (R) are omitted per standard practice, giving four governing combinations:

ID Formula Governing Load
LC-W11.2D + 1.6WWind dominant
LC-W20.9D + 1.0WWind + minimum gravity
LC-S11.2D + 1.0E + 0.2SSeismic + snow
LC-S20.9D + 1.0ESeismic + minimum gravity

where:

  • D = dead load: tray self-weight + cable weight (lbs/ft)
  • W = design wind force per linear foot (ASCE 7-22 §29.4)
  • E = seismic force per ASCE 7-22 §13.3.1, resolved into lateral (E_lat) and vertical (E_v = ±0.2 × SDS × D) components
  • S = snow load per linear foot of tray

The controlling combination is the one with the highest resultant force √(vertical² + horizontal²). Combinations with zero lateral load (W = 0 or E = 0) are flagged N/A. See also: Seismic Bracing Calculator, Wind Load Calculator, Support Span Calculator.

Dead Load (D)

Sum of tray self-weight and total cable weight per linear foot. Use the Support Span Calculator to obtain the cable weight from the cable schedule. Typical empty ladder tray: 2–6 lbs/ft; fully loaded tray: 10–30 lbs/ft.

Seismic forces are computed internally via ASCE 7-22 §13.3.1. Set SDS = 0 to exclude seismic loads (LC-S1 and LC-S2 will be N/A). See Seismic Bracing for detailed results.

Snow Load (S) — Optional

For outdoor installations in snowy climates: S = pg × tray width (ft) × Ce × Ct × Cs × Is (ASCE 7-22 Chapter 7). Use 0 for indoor or warm-climate installations.

Load Combinations Help

This tool computes ASCE 7-22 §2.3.2 LRFD load combinations for cable tray structural support design. It combines dead load, seismic, wind, and snow loads into four factored combinations and identifies which governs (controls).

Workflow:

  1. Enter the dead load D (tray + cable weight per foot).
  2. Enter site seismic parameters (SDS, SD1, risk category) and installation geometry. Set SDS = 0 for non-seismic sites.
  3. Enter wind parameters (speed, exposure, height, tray width, fill). Set wind speed = 0 for indoor/sheltered installations.
  4. Enter snow load if applicable. Use 0 for indoor or warm climates.
  5. Click Calculate Load Combinations.

The controlling combination (highest resultant force) is highlighted. Use that resultant to size support members and connection hardware.

For detailed seismic or wind results, use the dedicated Seismic Bracing and Wind Load pages.
