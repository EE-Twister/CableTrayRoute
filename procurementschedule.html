Ordered-Length Cable Procurement Schedule

Method & Assumptions

Cables are grouped by conductor specification (type + size + material). A field-trim tolerance (default 3%) is added to each cable's routed length per IEEE 1185 §6.4 to account for connector strain relief and termination offcuts.

The required cut lengths are assigned to standard reel sizes using a greedy first-fit-decreasing bin-packing algorithm. Cuts are placed on the first reel with sufficient remaining capacity; a new reel is opened only when no existing reel can accommodate the cut. A 5% residual threshold prevents reels from being used when only a tiny sliver remains.

References: IEEE Std 1185 §6.4, ICEA S-93-639, NEC Article 310.

See also: Pull Cards, Spool Sheets, Optimal Route.

Procurement Schedule Help

This tool generates an ordered-length cable procurement schedule from your routing results.

How it works: Cables are grouped by conductor specification (type + size + material). A configurable field-trim tolerance is applied to each cable's routed length. The resulting cut lengths are bin-packed into standard reel sizes to minimise waste.

Line items: Each row represents one cable specification. Click Show in the Cuts column to see the individual cable-level assignments with pull references.

Export: Use Export CSV to download the schedule for use in purchasing systems or spreadsheets.

Prerequisites: Route results must be present in the current session. Run the Optimal Route tool first.
