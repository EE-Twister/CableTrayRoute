CableTrayRoute Help

Browser-based toolkit for electrical scheduling, one-line modeling, analysis, routing, and deliverables. Use modules independently or follow the integrated workflow and reconcile schedule data explicitly when ready.

8-Step Workflow

  1. Equipment List Define major equipment tags, ratings, descriptions, and locations.
  2. Load List Capture loads, source relationships, demand factors, and operating data.
  3. One-Line Draw electrical relationships and use Reconcile Schedules to preview schedule updates.
  4. Cable Schedule Complete schedule-ready cable rows with tag, from/to, conductor size, and length.
  5. Raceway Schedule Catalog trays, conduits, and ductbanks for fill and routing.
  6. Fill / Routing Run tray fill, conduit fill, ductbank checks, and optimal routing.
  7. Studies Run demand, load flow, short-circuit, arc flash, TCC, harmonics, and motor studies.
  8. Deliverables Generate reports, pull cards, spool sheets, procurement schedules, estimates, and submittals.

Tips

  • Use Settings ⚙ in the nav to switch themes (Light, Dark, High Contrast) and units (Imperial / Metric).
  • Projects auto-save to your browser. Use Save Project to sync to the server or Export Project to download a .ctr.json backup.
  • The Progress Overview on this page tracks which steps have data and shows your next recommended action.
  • Most tables support CSV import and export — look for the import/export toolbar on each page.
