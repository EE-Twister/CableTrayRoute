Differential Protection (87B / 87T / 87G)

Method & Assumptions

Percentage differential protection follows IEEE C37.91-2008 (transformer protection), IEEE C37.102-2006 (generator protection), IEEE C37.97-2020 (bus protection), and IEC 60255-151:2009.

Operate and restraint currents (per-unit of rated):

I_op  = |I₁_pu − I₂_pu|          (phasor difference of normalised winding currents)
I_res = (|I₁_pu| + |I₂_pu|) / 2   (average — represents through-fault level)

Dual-slope trip characteristic (IEC 60255-151 §5):

Zone 1 (I_res ≤ I_res_break):  trip if I_op ≥ I_min + slope₁ × I_res
Zone 2 (I_res  > I_res_break):  trip if I_op ≥ I_min + slope₂ × I_res

Harmonic restraint blocking:

  • 2nd harmonic (inrush) — if |I₂nd| / |I₁fund| ≥ IHR₂ threshold, the relay is blocked. Typical threshold: 15–20%.
  • 5th harmonic (overexcitation) — if |I₅th| / |I₁fund| ≥ IHR₅ threshold, the relay is blocked. Typical threshold: 20–35%.

Winding correction (transformer): For delta-wye (Yd/Dy) connections, the secondary CT current is corrected for the 30° phase shift using a √3 magnitude factor and digital phase-rotation compensation (per IEEE C37.91-2008 §5.4).

Citations: Differential Protection Guide. See also: TCC / Protection Coordination, Short Circuit, Arc Flash.

IEC vector group of the transformer. Determines the magnitude correction factor (√3) and phase shift compensation applied to the secondary CT current.

Current Transformer (CT) Ratios

Primary winding CT ratio. E.g. 100:5 CT → enter 20 (100/5). Obtain from CT nameplate or protection relay settings sheet.

Secondary winding CT ratio. E.g. 600:5 CT → enter 120 (600/5).

Relay Settings

Minimum operate current in per-unit of rated. Typical range: 0.10–0.30 pu. Set higher for high-impedance transformers; lower for sensitive protection.

Slope of the percentage-differential characteristic in the low-restraint zone. Typical value: 0.20–0.30. Protects against CT mismatch errors on light through-faults.

Slope in the high-restraint zone (above the breakpoint). Typical value: 0.40–0.60. Higher slope improves stability during heavy through-faults with CT saturation.

Restraint current at which the characteristic transitions from slope 1 to slope 2. Typical value: 1.5–3.0 pu. Set to the expected heavy through-fault level.

Harmonic Restraint Thresholds

Block relay for inrush when 2nd harmonic content ≥ this fraction of fundamental. Typical setting: 0.15 (15%). Lower values = more sensitive inrush detection.

Block relay for overexcitation when 5th harmonic ≥ this fraction of fundamental. Typical setting: 0.20 (20%).

Differential Protection Help

This tool analyses percentage differential relay operation (87B, 87T, 87G) per IEEE C37.91-2008 and IEC 60255-151:2009.

Workflow

  1. Select the protected element type (transformer, bus, or generator).
  2. Enter nameplate data: MVA rating and HV/LV voltages.
  3. Enter CT ratios for each winding. Ensure CT ratings match the relay input range.
  4. Set relay thresholds: minimum pickup (Imin), slope 1, slope 2, and breakpoint.
  5. Enter per-phase primary and secondary currents (from fault study or field measurements).
  6. Click Run Analysis to see relay status and the Iop vs Ires plot.

Interpreting results

  • RESTRAINED — operating point is below the trip threshold. Relay does not operate. Normal for load and through-fault conditions.
  • OPERATE — differential current exceeds the percentage characteristic. Relay trips within 1–2 cycles. Indicates an internal fault.
  • HARMONIC_BLOCKED — the relay would operate but is restrained by 2nd (inrush) or 5th (overexcitation) harmonic content.
