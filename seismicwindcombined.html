Seismic + Wind Combined Load Scenario

Method & Assumptions

This tool orchestrates three ASCE 7-22 analysis modules to produce a unified combined-scenario result for cable tray structural support design:

  1. Seismic brace forces per ASCE 7-22 §13.3.1: 
    Fp = (0.4·ap·SDS·Wp / (Rp/Ip)) × (1 + 2z/h)
    Lateral force E = Fp × Wp; vertical Ev = ±0.2·SDS·Ip·Wp.
  2. Wind force per ASCE 7-22 §29.4: 
    q_z = 0.00256·Kz·Kzt·Ke·V²   (lbs/ft²)
W = q_z·G·Cf·(width × span) / span   (lbs/ft)
  3. LRFD load combinations per ASCE 7-22 §2.3.2: 
    LC-W1: 1.2D + 1.6W      (wind dominant)
LC-W2: 0.9D + 1.0W      (wind + minimum gravity)
LC-S1: 1.2D + 1.0E + 0.2S  (seismic + snow)
LC-S2: 0.9D + 1.0E      (seismic + minimum gravity)
    The governing (envelope) combination is identified by maximum resultant force.

Note: This page uses the §2.3.2 four-combination model. For LRFD/ASD toggle with six combinations and capacity utilization checks, see the Combined Loads (§2.3.1) page. Set wind speed to 0 for indoor or sheltered installations — wind combinations will be marked N/A.

See also: Seismic Bracing, Wind Load, Load Combinations.

Dead Load

Sum of tray self-weight and total cable weight per foot of tray. Use the Support Span Calculator for guidance. Typical empty ladder tray: 2–6 lbs/ft. Full tray: 10–30 lbs/ft.

Seismic Parameters

From ASCE 7 ground motion maps or USGS Seismic Design Maps. Typical range 0.1–2.5 g.

Mapped value S1 × Fv. Used to determine Seismic Design Category.

Per IBC 2021 Table 1604.5.

Wind Parameters

ASCE 7-22 Figure 26.5-1A (Risk Cat II). Set to 0 for indoor or fully sheltered installations — wind combinations (LC-W1, LC-W2) will be marked N/A.

= 1.0 for flat terrain. Increase for hilltop or escarpment sites (ASCE 7-22 §26.8).

Snow Load (optional — leave 0 for no snow)

Applied in LC-S1 only (factor 0.2S). Typically 0 for indoor installations and sites outside snow regions. Compute from ground snow load × tray width × exposure/thermal/slope factors (ASCE 7-22 Chapter 7).

Seismic + Wind Combined Help

This tool combines seismic and wind loads on cable tray supports and evaluates all four ASCE 7-22 §2.3.2 LRFD combinations to identify the governing demand.

Single tray mode: enter site seismic and wind parameters and tray weight, then click Calculate Combined Loads. The result table shows all four LRFD combinations with the governing row highlighted.

Schedule mode: reads all trays and cables from the current project and evaluates each tray section using the same site parameters.

Get SDS and SD1 values from the USGS Seismic Design Maps. Get basic wind speed from ASCE 7-22 Figure 26.5-1.

For LRFD/ASD six-combination analysis with capacity utilization checks, use the Combined Loads (§2.3.1) page.
