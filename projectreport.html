Presets pre-select the sections most commonly required for each deliverable type. Individual sections can always be toggled after applying a preset.

Report Package Builder Help

Select a preset to pre-configure sections, or toggle individual sections manually. Fill in the cover sheet and revision history, then click Generate Preview to build the report.

Export options:

  • Print / PDF — Opens the browser print dialog. Use "Save as PDF" for a file.
  • Export XLSX — Multi-sheet workbook (one sheet per section with tabular data).
  • Export HTML — Self-contained HTML file for sharing without a browser session.
  • Export JSON — Raw package data for integration with other tools.
  • Save Snapshot — Saves the current package to project storage for later reload.
