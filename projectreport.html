Unified Project Report

About This Report

The Project Report aggregates all analysis results into a single printable document. It covers:

  • Cable Schedule — all cables with routing status and lengths
  • Raceway Fill — tray and conduit utilisation vs NEC §392.22 limits
  • Clash Detection — 3D interference analysis (NEMA VE 2 §8.4)
  • Spool Sheets — prefabrication assembly groups (NEMA VE 2 §9)
  • Validation — unrouted cables, empty trays, missing data

Use Print / Save as PDF to produce the final deliverable.
