Egress & Emergency Lighting Calculator

Calculate average maintained illuminance using the lumen method and verify NFPA 101 §7.9.2 egress compliance (average ≥ 1 fc, minimum ≥ 0.1 fc).

Method & Standards

Lumen Method (IES HB-10): The average maintained illuminance on the workplane is calculated as:

Eavg = (N × F × CU × LLF) / A

SymbolParameterTypical value
NNumber of luminaires—
FInitial rated lumens per luminaireFrom IES file or manufacturer
CUCoefficient of UtilizationFrom CU table (RCR + reflectances)
LLFLight Loss Factor (maintained)0.80 typical for LED
ARoom area (ft²)L × W

Room Cavity Ratio (RCR): RCR = 5 × Hc × (L + W) / (L × W), where Hc = fixture mounting height − workplane height.

CU table: Based on a representative generic direct-component LED troffer/panel. For final designs, use the CU table from the manufacturer's IES photometric file.

Point-by-point grid: When fixture positions and an IES photometric file are provided, the tool calculates a 10×10 workplane illuminance grid using the cosine-cube method: E = I(θ) × cos(θ) / D².

Egress compliance (NFPA 101-2021 §7.9.2.1): Average illuminance ≥ 1.0 fc and minimum illuminance ≥ 0.1 fc along the path of egress.

Note: The CU table values are indicative. Results should be verified against the selected luminaire's manufacturer-supplied photometric data for final design documentation.

Egress Lighting Help

Enter the room dimensions, reflectances, and luminaire parameters, then click Calculate to get the average illuminance and NFPA 101 §7.9.2 egress compliance result.

Room Cavity Ratio (RCR): The tool calculates RCR automatically from the room dimensions and fixture mounting height. A higher RCR means more light is absorbed by wall surfaces before reaching the workplane, resulting in a lower CU.

Light Loss Factor (LLF): Accounts for luminaire lumen depreciation, dirt accumulation, and other maintenance-related losses. A value of 0.80 is typical for LED luminaires on a regular cleaning and relamping schedule.

IES file (optional): Upload an ANSI/IES LM-63 photometric file for your luminaire to enable point-by-point grid calculation. Enter individual fixture positions (x, y from the room corner) after loading an IES file.

NFPA 101 §7.9.2.1: Emergency means of egress lighting must provide not less than an average of 1.0 fc (10.8 lux) and not less than 0.1 fc (1.1 lux) at any point along the path of egress.
